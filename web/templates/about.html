Overview

Platforms such as Google Scholar and Web of Science allow users to search for authors, search by paper, and see how many citations a given paper has. The goal of this project is to create a dashboard-style interface that displays activity and engagement surrounding scientific publications. Rather than focusing on numbers of citations, we will be displaying the number of interactions. The data describing these interactions is taken from the Crossref API ↗.
