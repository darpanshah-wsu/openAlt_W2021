Overview

The goal of this project is to create a dashboard-style interface that displays activity and engagement surrounding scientific publications.
Crossref is an official agency responsible for assigning DOIs (Document Object Identifiers) to published papers.

The Crossref Event API provides information on how and where scholarly articles are discussed. The goal of this interface is to organizes, simplify, and visualize this Event data.

This project is open source. All the code and documentation is available on GitHub.
